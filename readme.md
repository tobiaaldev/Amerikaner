# Amerikaner Card Game (Java)

A Java implementation of the trick-taking card game **"Amerikaner"** using Monte Carlo simulation for move evaluation

## Features
- Game state management
- Trick resolution and scoring
- Monte Carlo based AI player for move avalutation

## Current status
- Core game logic implemented
- Simulations seem to be working
- Bidding phase not yet implemented
